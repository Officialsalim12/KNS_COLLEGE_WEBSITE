Back End Web Development

Learn Server-Side Development with MySQL and PostgreSQL. Build Robust Database-Driven Applications

Programme Summary

This comprehensive certificate program teaches you server-side web development and database management. Learn to build robust back-end systems using MySQL and PostgreSQL databases. You'll master database design, SQL queries, API development, and server-side programming. Through hands-on projects, you'll create database-driven applications and RESTful APIs that power modern web applications. This program is perfect for developers looking to specialize in back-end development or professionals wanting to understand how server-side systems work.

Programme Highlights

Database Mastery

Master MySQL and PostgreSQL database systems.

API Development

Build RESTful APIs and server-side applications.

Server-Side Programming

Learn back-end development and server architecture.

Real-World Projects

Build database-driven applications from scratch.

Programme Details

Duration: 10 Weeks
Study Time: 10-15 hours per week
Language: English
Delivery Mode: Hybrid
Credential: Certificate of Completion
Intake Periods: Rolling Enrollment
Assessment: Projects (80%), Exams (20%)

Learning Outcomes

Upon successful completion of this certificate, you will be able to:

  • Design and create relational databases using MySQL and PostgreSQL.
  • Write complex SQL queries to retrieve and manipulate data.
  • Build RESTful APIs and server-side applications.
  • Implement database security and best practices.
  • Optimize database performance and queries.
  • Handle database transactions and data integrity.
  • Connect databases to web applications.
  • Deploy and manage database-driven applications.

Programme Structure

The program is delivered over 10 weeks through hybrid sessions combining online learning with instructor-led workshops.

Core Modules

  • Introduction to Databases & SQL
  • MySQL Fundamentals & Administration
  • PostgreSQL Fundamentals & Advanced Features
  • Database Design & Normalization
  • Advanced SQL Queries & Stored Procedures
  • API Development & RESTful Services
  • Server-Side Programming
  • Database Security & Performance Optimization
  • Project: Build a Database-Driven Application
