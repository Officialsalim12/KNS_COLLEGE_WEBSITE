Programme Summary

This comprehensive certificate program teaches you the fundamentals of front-end web development. Learn to create beautiful, responsive, and interactive websites using HTML, CSS, and JavaScript. You'll master modern front-end development techniques, including responsive design, CSS frameworks, JavaScript ES6+, and DOM manipulation. Through hands-on projects, you'll build real-world websites and web applications that showcase your skills. This program is perfect for beginners looking to start a career in web development or professionals wanting to add front-end skills to their toolkit.

Programme Highlights

Modern Technologies

Learn HTML5, CSS3, and modern JavaScript (ES6+).

Responsive Design

Master responsive web design for all devices.

Interactive Websites

Build dynamic, interactive user interfaces with JavaScript.

Portfolio Projects

Create a portfolio of real-world projects.

Programme Details

Duration: 8 Weeks
Study Time: 10-15 hours per week
Language: English
Delivery Mode: Hybrid
Credential: Certificate of Completion
Intake Periods: Rolling Enrollment
Assessment: Projects (80%), Exams (20%)

Learning Outcomes

Upon successful completion of this certificate, you will be able to:

  • Write semantic HTML5 markup for web pages.
  • Style websites using CSS3, including Flexbox and Grid layouts.
  • Create responsive designs that work on all screen sizes.
  • Write JavaScript code to add interactivity to websites.
  • Manipulate the DOM to create dynamic user experiences.
  • Use modern JavaScript features (ES6+) including arrow functions, promises, and async/await.
  • Build interactive web applications from scratch.
  • Deploy websites to the web using modern hosting platforms.

Programme Structure

The program is delivered over 8 weeks through hybrid sessions combining online learning with instructor-led workshops.
