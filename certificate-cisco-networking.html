Programme Details

Duration: 8 Weeks
Study Time: 10-15 hours per week
Language: English
Delivery Mode: Hybrid
Credential: Certificate of Completion
Intake Periods: Rolling Enrollment
Assessment: Projects (80%), Exams (20%)

Learning Outcomes

Upon successful completion of this certificate, you will be able to:

  • Write semantic HTML5 markup for web pages.
  • Style websites using CSS3, including Flexbox and Grid layouts.
  • Create responsive designs that work on all screen sizes.
  • Write JavaScript code to add interactivity to websites.
  • Manipulate the DOM to create dynamic user experiences.
  • Use modern JavaScript features (ES6+) including arrow functions, promises, and async/await.
  • Build interactive web applications from scratch.
  • Deploy websites to the web using modern hosting platforms.

Programme Structure

The program is delivered over 8 weeks through hybrid sessions combining online learning with instructor-led workshops.
